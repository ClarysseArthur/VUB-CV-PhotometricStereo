# Dataset
The dataset included in the submission is part of the the Harvard Photometric Stereo Dataset.
We selected the cat subset to keep the file size low; the original source also includes datasets for frogs, lizards, and more.

> Xiong, Ying, Ayan Chakrabarti, Ronen Basri, Steven J. Gortler, David Jacobs, and Todd Zicker. n.d. “From Shading to Local Shape.” From Shading to Local Shape. Accessed December 4, 2025. https://vision.seas.harvard.edu/qsfs/index.html.
